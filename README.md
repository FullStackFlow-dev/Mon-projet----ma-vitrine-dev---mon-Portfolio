#   Projet Portfolio Professionnel – Merph-dev

Ceci est ma carte de visite numérique et une vitrine de mon développement professionnel autodidacte.  
Ce projet centralise mes compétences, mes projets et mes certifications dans les domaines du développement web, de la cybersécurité, du data engineering et de l’intelligence artificielle.



##   Objectifs du projet

- Présenter mes compétences techniques de manière claire et structurée.
- Illustrer ma progression à travers des projets concrets et documentés.
- Offrir une vitrine professionnelle pour les recruteurs, collaborateurs et partenaires.
- Mettre en pratique un workflow Git rigoureux et pédagogique.



##   Technologies utilisées

- **Front-End** : HTML5, CSS3, JavaScript, React, Vue, Angular  
- **Back-End** : Node.js, Python (Flask, Django), PHP, Java (Spring Boot)  
- **Bases de données** : MongoDB, PostgreSQL, MySQL  
- **Cybersécurité** : Nmap, Wireshark, Metasploit, Burp Suite  
- **Data** : Apache Airflow, Spark, Power BI, Pandas  
- **IA & LLM** : GPT, LangChain, OpenAI API, Hugging Face  
- **Outils** : Git/GitHub, Docker, CI/CD, VS Code, Linux



##   Structure du projet

- `index.html` : structure principale du portfolio  
- `sections/` : chaque section du site (À propos, Compétences, Projets, etc.)  
- `assets/` : images, logos, captures de projets  
- `README.md` : documentation du projet  
- `LICENSE` : licence d’utilisation ()



##   Workflow Git utilisé

Chaque section est développée dans une branche dédiée :

- `feature-accueil` → Accueil  
- `feature-navigation` → Menu  
- `feature-a-propos` → À propos
- `feature-a-blog` → Blog
- `feature-competences` → Compétences  
- `feature-projets` → Projets  
- `feature-contact` → Formulaire de contact  
- `feature-footer` → Pied de page
